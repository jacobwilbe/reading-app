# DailyReaderIOS (SwiftUI)

iOS Xcode version of the Daily Reader MVP.

## Open

- Open `/Users/jacobwilber/Documents/New project/reading-app-ios/DailyReaderIOS.xcodeproj` in Xcode.
- Choose an iPhone simulator.
- Press Run.

## Included screens

- Plan (onboarding: minutes + topic/random mode)
- Daily (recommendation list)
- Reading (article detail + mark as read)
- Weekly (wrapped metrics + suggestions)

## Notes

- Current data is in-memory mock data (`AppStore.swift`).
- Recommendation scoring mirrors the Next.js version.
- Next step is replacing mock data with Supabase-backed API calls.
